Object deep sort
=========

A small library that sorts object by keys and arrays

## Installation

  `npm install sort-deep-object`

## Usage

    var sort = require('sort-deep-object');

    var sortedObject = sort({b:2, a:1});
  
  
  Output should be `{a:1, b:2}`
